# 📋 Formulário de Cadastro

Formulário de cadastro desenvolvido com HTML5, CSS3 e JavaScript, com o objetivo de demonstrar a criação de formulários interativos e a aplicação de validações básicas no front-end. Interface simples, clara e funcional, projetada para coletar informações essenciais do usuário de forma organizada e intuitiva.

🔗 **Demo:** [franciellyvicente.github.io/Page](https://franciellyvicente.github.io/Page/)

## 🛠️ Tecnologias

- HTML5 – estrutura e semântica do formulário
- CSS3 – estilização e organização do layout
- JavaScript – validação dos campos e lógica de verificação de senha

## ✨ Funcionalidades

- Campos obrigatórios com validação (nome, sobrenome, e-mail, senha e confirmação de senha)
- Verificação de correspondência entre senha e confirmação de senha
- Limite mínimo e máximo de caracteres
- Seleção de gênero com múltiplas opções (Mulher, Homem, Outros, Prefiro não informar)
- Interface simples, clara e responsiva

## 🎨 Design

- Fundo preto, com o formulário em branco e detalhes em laranja

## 🧾 Campos do formulário

**Obrigatórios:**
- Nome
- Sobrenome
- E-mail
- Senha
- Confirmação de senha

**Gênero (seleção):**
- Mulher
- Homem
- Outros
- Prefiro não informar

## ⚙️ Validações implementadas

- Verificação de campos obrigatórios
- Limite mínimo e máximo de caracteres
- Validação de dados digitados pelo usuário
- Verificação de correspondência entre senha e confirmação de senha

Essas validações ajudam a evitar erros de preenchimento e contribuem para a integridade dos dados antes do envio ao back-end.
